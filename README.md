Название: Определитель сигналов

Краткое описание:
Веб-приложение необходимо для орпеделиения инопланетной цивилизаци путём обработки 
звука из сигнала, который они прислали, при помощи обучаемой нейросети

Требования:
Иметь Pycharm, установить следующие библиотеки: Flask, flask_sqlalchemy,
numpy, tensorflow

Установка:
Для установки нужно зайти по ссылке https://github.com/rrrrde1go/PREDPROFF и
скопировать репозиторий себе на устройство, затем запустить app.py 

Использование:
На вход подаётся 2 массива для обучения нейронной сети, затем массив с посланиями
и массив с номером цивилизации для валидации
